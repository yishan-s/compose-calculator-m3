# Project Name
> 使用 Jetpack Compose 與 Material 3 設計的計算機，練習架構與狀態管理。

## Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Architecture:** MVVM / Clean Architecture (Hopefully)
* **Libraries:** Retrofit, Room, Hilt, Coil...

## Project Status
- [x] Project Initialization (Empty Activity)
- [ ] UI Development (Material 3 Calculator Layout)
- [ ] Calculator Logic Implementation (Arithmetic Operations)
- [ ] State Management with ViewModels
- [ ] History Management (Room Database integration)
- [ ] Interactive History (Re-edit from past entries)

## Key Features
- **Essential Arithmetic:** Supports basic addition, subtraction, multiplication, and division with a clean Material 3 interface. 
- **Calculation History:** Automatically saves calculations to a local database using Room. 
- **Time-Travel Editing:** Similar to the iOS calculator experience, users can tap into past calculations from the history list to re-edit or continue operations. 
- **Adaptive Theme:** Full support for Material You (Dynamic Color) and Dark Mode.

## Quick Start
1. **Clone this Repository**
```
git clone https://github.com/yishan-s/compose-calculator-m3.git
```
2. **Open the project** in Android Studio. 
3. **Sync Gradle** and ensure you have the correct version configured. 
4. **Run the app** on a physical device or an emulator (API Level 26+).
